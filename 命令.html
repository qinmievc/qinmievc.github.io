mysql

1、mysql服务的启动和停止
net stop mysql net start mysql
2、登陆mysql
mysql -u root -p
3、增加新用户
grant 权限 on 数据库.* to 用户名@登录主机 identified by "密码"
4.选择数据库
use 数据库名
5.显示数据库列表
show databases;
6.显示数据表的结构
describe 表名;
7.建库与删库
create database 库名（character set utf8）;
drop database 库名;
8.建表与删表
use 库名;
create table 表名(字段列表);
drop table 表名;

django

1.创建项目
django-admin startproject 项目名称
2.创建APP
python manage.py startapp APP名称
或django-admin startapp APP名称
3.创建数据库表或更改数据库表或字段
# 1. 生成新的迁移文件
python manage.py makemigrations
# 2. 将生成的py文件映射应用到数据库
python manage.py migrate
4.使用开发服务器——启动：
python manage.py runserver
5.清空数据库
python manage.py flush
6.创建超级管理员
python manage.py createsuperuser
# 按照提示输入用户名和对应的密码就好了邮箱可以留空，用户名和密码必填
# 修改 用户密码可以用：
python manage.py changepassword username

conda

1.创建环境
conda create -n env-name python=3.6
2.删除环境
conda env remove -n env-name
3.列出所有环境
conda env list (conda list 查看conda下的包)
4.升级conda
conda update conda
5.升级所有包
conda update --all
6.conda升级后释放空间
conda clean -p删除没有用的包
conda clean -t删除保存下来的压缩文件（.tar）
7.进入环境
conda activate 环境
8.退出环境
conda deactivate 环境
9.安装库
pip(或conda) install 库 （..库==可查看历代版本，uninstall为删除）
10.查看已安装库
pip list / pip freeze
11.显示指定包
pip show numpy / pip freeze | findstr numpy （conda search search-term，可以模糊搜索）
12.把环境添加到jupyter notebook
pip install ipykernel
python -m ipykernel install --name 名字 方便起见建议统一
13.查看已添加到jupyter notebook的kernel
jupyter kernelspec list
14.删除指定的kernel
jupyter kernelspec remove 名字
15.删除环境所有包
conda remove --name hk --all
16.测试是否是GPU
import torch
flag = torch.cuda.is_available()
print(flag)
